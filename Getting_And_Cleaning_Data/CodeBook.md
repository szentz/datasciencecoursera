# Code Book

## Logic flow of `run_analysis.R`
1. Download and unzip the data if needed.
2. Load the activity data
3. Loads the feature info
4. Loads both the training and test datasets.  Filter data to keep only columns with a mean or standard deviations.
5. Loads the activity and subject data. 
6. Merges the training data with the subject and activity data.  Repeat for test data.
7. Merge test and training data
8. Converts the `activity` and `subject` columns into factors
9. Creates a tidy dataset, `tidy.txt`, containing the mean value of each
   variable for each subject and activity pair.


## Variables of `run_analysis.R`
`trainData`, `testData`, `trainSubjects`, `testSubjects`, `trainActivities`, `testActivities` all contain download data.
`fullData` is the complete merged dataset of the training and test data with the subjects and activities.
`activityLabels` contains the names of the activities recorded
`features` contains the filtered column names
`fullDataMelt` and `fullDataMean` contain the reshaped data.

## Activity Labels

* `WALKING` (value `1`): subject was walking during the test
* `WALKING_UPSTAIRS` (value `2`): subject was walking up a staircase during the test
* `WALKING_DOWNSTAIRS` (value `3`): subject was walking down a staircase during the test
* `SITTING` (value `4`): subject was sitting during the test
* `STANDING` (value `5`): subject was standing during the test
* `LAYING` (value `6`): subject was laying down during the test
